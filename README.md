## Running GrowEverything Locally

### Manual Setup

Create a python virtual environment, install the dependencies and run the migrations
```bash
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
python3 manage.py runserver
```

Run the server
```bash
python3 manage.py runserver
```
